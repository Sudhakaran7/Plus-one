Given a non-empty array of digits representing a non-negative integer, increment one to the integer.
The digits are stored such that the most significant digit is at the head of the list, and each element in the array contains a single digit.
You may assume the integer does not contain any leading zero, except the number 0 itself.

Input Description:
First line contains, N the size of the array.(1<N<100)
Second line contain, N elements of integers in the array.

Output Description:
Print the digits after incrementing by 1.

Sample Input:
1 2 3

Sample Output:
1 2 4

Explanation:
The given number is 123 so incrementing by 1, the output is 124
